# Timer Icon

Place your timer icon here (PNG format, 24x24 or 32x32 pixels recommended).

Filename: `timer-icon.png`

If no custom icon is provided, the application will use the system's "alarm-clock" themed icon.

You can find free icons at:
- https://iconmonstr.com/
- https://www.flaticon.com/
- https://remixicon.com/
